**Smart File Manager & Automator** is a lightweight, visually appealing Windows Forms application built with C# to help you instantly reclaim control over cluttered directories like Downloads or Desktop.

Instead of manual sorting, this tool automates your digital organization with a single click, featuring a modern flat-design UI and a real-time terminal-style logger.

---

## ✨ Key Features

* 🗂️ **Smart Categorization:** Automatically sorts files into logical categories (Images, Documents, Videos, Archives, etc.) based on their extensions.
* 📅 **Time-Based Archiving:** Organizes files into a clean `Year \ Month` folder structure derived from their creation dates.
* 🧹 **Deep Clean Engine:** Scans the target directory and eliminates all redundant empty folders, leaving your workspace spotless.
* 💻 **Terminal-Style Logging:** Features a built-in dark mode log viewer to track every file move and system change in real-time.
* 🚀 **Clean Architecture:** Procedurally built UI and business logic in a single-file structure for maximum portability and ease of learning.

---

## 🛠️ Technical Stack

- **Language:** C#
- **Framework:** .NET (Windows Forms)
- **IO Logic:** System.IO (Advanced Directory/File manipulation)
- **UI Style:** Modern Flat UI with Terminal Emulation

## 🚀 Getting Started

### Prerequisites
* Visual Studio 2019 or newer.
* .NET Framework 4.7.2+ or .NET 6/7/8.

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/SmartFileManager.git](https://github.com/your-username/SmartFileManager.git)
    ```
2.  **Open the Project:**
    Launch the `.sln` file in Visual Studio.
3.  **Build and Run:**
    Press `F5` to start organizing your files.

---

## 📝 How It Works

1.  **Select Folder:** Pick the cluttered directory you want to organize.
2.  **Choose Strategy:** Select between extension-based sorting or time-based archiving.
3.  **Automate:** Click the **Start Organizing** button.
4.  **Monitor:** Watch the real-time terminal log as the application moves files and cleans up directories.
