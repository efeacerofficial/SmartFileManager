# ⚡ Smart File Manager & Automator

A lightweight, visually appealing Windows Forms application built with C# to help you instantly organize cluttered directories like your Downloads or Desktop.

Instead of dealing with messy folders manually, this tool automates the process with a single click, featuring a modern, flat-design UI and a real-time terminal logger.

## ✨ Features

* 🗂️ **Smart Categorization:** Automatically sorts files into predefined categories (Images, Documents, Videos, Archives, etc.) based on their file extensions.
* 📅 **Time-Based Sorting:** Organizes files logically into a `Year \ Month` folder structure based on their creation dates.
* 🧹 **Deep Clean:** Scans the target directory and permanently deletes all leftover empty folders.
* 💻 **Terminal-Style Logging:** Watch every file move and directory change in real-time through the built-in dark mode log viewer.
* 🚀 **Standalone & Clean Architecture:** The entire UI and business logic are built procedurally in a single file, making it incredibly easy to read, modify, and learn from.
